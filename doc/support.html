Support¶

I am using Google Code for support services. Currently these are:

  • A mailing list / discussion group.
  • An issue tracker.
  • Source code repository (a regularly updated clone of the entire tree, but not my main repo).

To ask questions, report bugs, and generally discuss Lepl, please post to the group.
